This guide walks you through training a model using the example dataset included with every new Onyx Engine account.

<Info>
Make sure you've completed the [installation](/installation) steps before continuing.
</Info>

## Train a Model

Create a new Python file and add the following code:

```python
from onyxengine import Onyx
from onyxengine.modeling import (
    Output,
    Input,
    MLPConfig,
    TrainingConfig,
    AdamWConfig,
)

# Initialize the client (uses ONYX_API_KEY env var)
onyx = Onyx()

# Define model outputs and inputs
outputs = [
    Output(name='acceleration_predicted'),
]
inputs = [
    Input(name='velocity', parent='acceleration_predicted', relation='derivative'),
    Input(name='position', parent='velocity', relation='derivative'),
    Input(name='control_input'),
]

# Configure the model
model_config = MLPConfig(
    outputs=outputs,
    inputs=inputs,
    dt=0.0025,
    sequence_length=8,
    hidden_layers=3,
    hidden_size=64,
    activation='relu',
    dropout=0.2,
)

# Configure training
training_config = TrainingConfig(
    training_iters=2000,
    train_batch_size=1024,
    optimizer=AdamWConfig(lr=3e-4, weight_decay=1e-2),
)

# Start training
onyx.train_model(
    model_name='my_first_model',
    model_config=model_config,
    dataset_name='example_train_data',
    training_config=training_config,
    monitor_training=True
)
```

Run the script to start training. The progress displays in your terminal:

```
Preparing to train model [my_first_model] using dataset [example_train_data].
Training started...
[====================================] 100% | 2000/2000 iters | Loss: 0.0023
Training complete!
```

## Load and Use the Model

Once training completes, load your model and run a simulation:

```python
import torch
from onyxengine import Onyx

# Initialize the client
onyx = Onyx()

# Load the trained model
model = onyx.load_model('my_first_model')

# Create initial state tensors
batch_size = 1
seq_length = model.config.sequence_length

# Initial conditions (velocity and position)
velocity = torch.zeros(batch_size, seq_length, 1)
position = torch.zeros(batch_size, seq_length, 1)

# External inputs for the full simulation
sim_steps = 100
control_input = torch.ones(batch_size, seq_length + sim_steps, 1) * 0.5

# Run simulation
result = model.simulate(
    x0={'velocity': velocity, 'position': position},
    external_inputs={'control_input': control_input},
    sim_steps=sim_steps
)

# Access results
print("Velocity trajectory shape:", result.inputs['velocity'].shape)
print("Position trajectory shape:", result.inputs['position'].shape)
print("Acceleration output shape:", result.outputs['acceleration_predicted'].shape)
```

## Understanding the Code

### Model Inputs and Outputs

The key to Onyx models is defining the relationships between features:

- **`Output`**: Features the model predicts (here, acceleration)
- **`Input`**: Features fed into the model, including:
  - **States**: Inputs derived from outputs using `parent` and `relation`
  - **External inputs**: Known values like control signals

The `relation` parameter defines how states update during simulation:
- `'derivative'`: The parent is the time derivative of this feature
- `'delta'`: The parent is the change per timestep
- `'equal'`: The parent equals this feature

### Simulation

The `simulate()` method handles multi-step prediction:

1. Takes initial state values in `x0`
2. Takes external inputs for the full trajectory in `external_inputs`
3. Automatically integrates states using the defined relations
4. Returns a `SimulationResult` with trajectory data for all features
